# Construction Company Landing Page

A React landing page for a construction company website. Includes sections for home, about, services, projects, contact, and footer with newsletter signup.

## Features

- Responsive landing page layout
- Hero section with company branding
- Services and project showcasegit 
- Contact form with localStorage persistence
- Newsletter signup
- Simple navigation links

## Prerequisites

- Node.js
- npm or yarn

## Install

```bash
cd "C:/Users/HP/Desktop/MY PROJECT/construction-company-main"
npm install
```

## Run

```bash
npm start
```

Then open the development server in your browser.

## Project Structure

- `src/spa/landing_page.jsx` — main landing page component
- `src/assets/` — images and icons used in the page
- `README.md` — project documentation

## Notes

- The contact form saves data to `localStorage` using the key `contactdata`.
- Newsletter form uses the same submit handler as the contact form.
